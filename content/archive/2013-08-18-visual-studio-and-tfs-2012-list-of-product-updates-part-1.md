## <font size="3">Unsure which version and update you have  ?</font>

  <font size="2">**Visual Studio:**</font>

  <font size="2">Go to the top menu, choose Help | About</font>

  [![image](/images/2015/08/GWB-Windows-Live-Writer-Visual-Studio-and-TFS-2012-product-insta_F68B-image_thumb2.png)](https://gwb.blob.core.windows.net/terje/Windows-Live-Writer/Visual-Studio-and-TFS-2012-product-insta_F68B/image_2.png)

  <font size="2">Note that both the name of the update and the corresponding version number is given.</font>

  <font size="2">Go to the top menu, select Tools | Extensions and Updates. Under the Updates | Product Updates section you will see if there is a new update ready for you:</font>

  [![image](/images/2015/08/GWB-Windows-Live-Writer-Visual-Studio-and-TFS-2012-product-insta_F68B-image_thumb_22.png)](https://gwb.blob.core.windows.net/terje/Windows-Live-Writer/Visual-Studio-and-TFS-2012-product-insta_F68B/image_6.png)

  (Image from VS 2013 preview, but it is the same in VS 2012)

  <font size="2">To the far right you see the version number you have, and what version to update to.  Since the updates are cumulative, you don’t need to worry about those in between.</font>

  <font size="2">**Team Foundation Server (on premise):**</font>

  <font size="2">Use the web access for the TFS. Go to the Help Icon to the far right, and choose “About” from the menu:</font>

  [![image](/images/2015/08/GWB-Windows-Live-Writer-Visual-Studio-and-TFS-2012-product-insta_F68B-image_thumb_12.png)](https://gwb.blob.core.windows.net/terje/Windows-Live-Writer/Visual-Studio-and-TFS-2012-product-insta_F68B/image_4.png)

  <font size="2">In this case I have a TFS 2013 preview server.</font>

  <font size="2">**Team Foundation Service:**</font>

  <font size="2">If you are connected to the  Team Foundation Service you will not find any version number.  It is also always updated to the latest version.  You could check the [service status page](http://tfs.visualstudio.com/en-us/support/current-service-status/) to see if all is working, and to see what has been done at </font><font size="2">the </font>[updates and release notes listed here](http://tfs.visualstudio.com/news/)<font size="2">.</font>
